# Taskify
A modern and responsive To-Do List application designed to help users organize their daily tasks efficiently. This project allows users to add, edit, delete, and mark tasks as completed with a clean and user-friendly interface.
# Features
Add new tasks easily
Edit existing tasks
Delete tasks
Mark tasks as completed
Responsive design for mobile and desktop
Simple and clean user interface
